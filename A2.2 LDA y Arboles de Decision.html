Importamos los datos de una nueva base de datos obtenida de UC Irvine Machine Learning Repository, donde queremos predecir si un hongo es venesoso o no. Esto se determina en la columna de class_num, donde un 1 indica que si es venenoso y por el contrario el 0 indicaria que no lo es. Despus de aplicar one-hot encoding, cada posible valor se convirti en una columna booleana. Despues, dividimos en train (80%) y test (20%) con el mismo balance de datos que se muestra a continuacion junto a las dimensiones.

# Generar objeto de validacin cruzada
skf = StratifiedKFold(n_splits=5)

# Generar secuencia de valores
ccp = np.linspace(0.001, 0.2, 250)

# Inicializar variable de salida
cv_scores = []

# Ciclo donde se calcula el f1-score mediante validacin cruzada
for alpha in ccp:
      pruned_tree = DTC(ccp_alpha=alpha)
      cv_scores.append(np.mean(cross_val_score(pruned_tree, X_train, y_train, cv=skf, scoring='f1')))
# Encontrar el alpha ptimo
alpha = ccp[np.argmax(cv_scores)]

# Imprimir resultados
print("Best alpha:",alpha)
# Generar y entrenar el rbol podado
pruned_tree = DTC(ccp_alpha=alpha).fit(X_train, y_train)
